## 📁 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/Dani2003G/atividade_avaliativa_1
# Entrar no diretório
$ cd atividade_avaliativa_1

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start

```
